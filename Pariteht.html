  erilaisten olioiden yhteistoiminnalla.
  Olio-ohjelmointia on kahta eri tyyppiä: Luokkapohjainen ja prototyyppipohjainen:
     -Luokkapohjaisessa ohjelmoinnissa oliot saavat toiminnallisuuden ja tilan luokan jäsenien mukaan.
     -Prototyyppiohjelmoinnissa oliot saavat toimintansa prototyyppioliolta.

Keskeiset sisäänrakennetut oliot

Keskeiset sisäänrakennetut oliot (Key built-in objects) ovat selaimissa itsestääntoimivia olioita, jotka ovat saatavilla riippumatta selaimen lataamasta sivusta.
Näitä sisäänrakennettuja olioita ovat:

  • -Array olio
  • -Boolean olio
  • -Date olio
  • -Math olio
  • -Number olio
  • -String olio
  • -RegExp olio

Olion ominaisuudet

Olion ominaisuuksilla (Object properties) viitataan olion jälkeen tulevaan muuttujaan. Ominaisuuksia voi lisätä ja poistaa aina tarvittaessa. Ominaisuuksiin voi viitata 2 eri tavalla:
-olionNimi.ominaisuus
-olionNimi["ominaisuus"] 

 
  const oppilas = {
    nimi: "Jaakko",
    ika: 18,
    silmienvari: "sininen"
  }

  // Lisää ominaisuuden
  oppilas.arvosana = "10";

  // Poistaa ominaisuuden
  delete oppilas.ika

  var teksti = document.getElementById("ominaisuuksia").innerHTML =

Konstruktorit

Konstruktori (Constructor) on metodi jolla alustetaan luokan muuttujat. 


  class Visitor {
    constructor(name) {
      this.name = name;
    }
    present() {
      return 'Name is ' + this.name;
    }
  }
  
  class Employee extends Visitor {
    constructor(name, age) {
      super(name);
      this.age = age;
    }
    show() {
      return this.present() + ', Age ' + this.age;
    }
  }
  
  visitor = new Visitor("Jussi");
  console.log(visitor.present());
  // Output: Name is Jussi
  
  employee = new Employee("Jaska",20);
  console.log(employee.show());
  // Output: Name is Jaska, Age 20

Kapselointi

Kapselointi (Encapsulation) on tiedon lukitsemista jotta sitä ei voi enää muokata myöhemmin. Kapselointi saadaan aikaan risuaita merkillä. 


  class Auto {
    #age;
    constructor(model, color) {
      this.model = model;
      this.color = color;
      // Auton ikä on # merkitty, joka tarkoittaa että sen arvoa ei voi muuttaa
      this.#age = this.getYear();
    }
    setColor(color) {
      this.color = color;
    }
    getColor() {
      return this.color;
    }
    getYear() {
      return 2023;
    }

    getAll() {
      console.log(this.model);
      console.log(this.color);
      console.log(this.#age);
    }
  }

  mersu = new Auto("mersu", "green");
  mersu.getAll();
  // Output: mersu, green, 2023

  mersu.color = "white";
  mersu.getAll();
  // Output; mersu, white, 2023

  mersu.age = 1234;
  mersu.getAll();
  // Output: mersu, white, 2023
  // mersu.age = 1234; Ei pysty vaihtamaan jo valmiiksi annettua arvoa 2023

Periyttäminen

Periyttäminen (Inheritance) on luokan jatkamista. Eli luokkaa voidaan periydyttää toiseen luokkaan, joka perii perityn luokan valmiiksi julistetut muuttujat mukanaan. Tähän uuteen jatkettuun luokkaan voidaan lisätä myös omia muuttujia. extendausta, eli luokkaa jatketaan extendaamisella toiseen luokkaan, viitaten; Puhelin, Väri, Koko. 

 
//Periytymisellä pystyy käyttämään koodia uudelleen, lisätä siihen uusia luokkia ja muokata sitä.
class Puhelin {
  constructor(merkki) {
    this.merkki = merkki;
  }
  malli() {
    return "Minulla on " + this.merkki;
  }
}

// Luokka Väri on jatkoa luokalle Puhelin
class Väri extends Puhelin {
  constructor(merkki, väri) {
    super(merkki);

    // Väri luokka lisää värin puhelimen merkkiin
    this.väri = väri;
  }
  vari() {
    return this.malli() + ", se on " + this.väri;
  }
}

// Koko lisää ko'on puhelimen värille ja merkille
class Koko extends Väri {
  constructor(merkki, väri, koko) {
    super(merkki, väri);
    this.koko = koko;
  }
  kaikki() {
    return this.malli() + ", se on " + this.väri + ". Sen koko on " + this.koko;
  }
}

let kokoPuhelin = new Koko("Samsung", "vihreä", "29cm²");

console.log(kokoPuhelin.malli());  
// Output: Minulla on Samsung

console.log(kokoPuhelin.vari());   
// Output: Minulla on Samsung, se on vihreä

console.log(kokoPuhelin.kaikki()); 
// Output: Minulla on Samsung, se on vihreä. Sen koko on 29cm²
